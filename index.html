MyShowcase™

An open source web application which offers personalised evidence gathering for e-portfolios

Users can gather, map and showcase content from a range of online sources, including social network spaces, to support learning and personal development. MyShowcase offers brand new possibilities for the e-portfolio in the age of social networking and even plugs straight into your VLE or LMS.

A bit about MyShowase...

MyShowcase - an open-source tool that opens up the value of social networks for learning and development, bridging the gap between personal spaces and e-portfolios.

Originally developed as part of a University of Hull initiative supported by JISC funding, development was contracted and undertaken as part of this project by MyKnowledgeMap Ltd.

Called CPD-Eng (Personalised systems supporting IPD and CPD within a professional framework), the project has attracted a lot of attention and a number of educational institutions undertook pilots using the tool with students during the 2010/11 academic year.

MyShowcase is Java-based. All code developed as part of this project is provided under a GNU Affero General Public License as open source software.

Features...

Gather

Open up brand new evidence streams for the e-portfolio with RSS feeds, Twitter, Flickr, learner-owned digital files, and weblinks.

Organise using tags

All evidence brought into the tool can be tagged it using the free-text tagger. Tags allow the user to group related evidence streams and easily access them in future.

Map to frameworks

Users can map their evidence to frameworks relevant to their study, including competency frameworks, lists of learning objectives, qualification frameworks, and Continuing Professional Development (CPD) frameworks.

Create showcases

Users can create showcases by hand-picking evidence to be made available to both internal and external reviewers, for use in assessment submissions, personal tutorials, portfolios for potential employers, and ongoing CPD.

Technical information

Core application

This download provides the latest available source code for the core MyShowcase application along with all required install files and build instructions. A pre-built WAR file is also included for quick set-up.

  • Tomcat 5.5.31+
  • Java 1.5 or 1.6
  • MySQL 5.1
  • MySQL Connector / J5.0.8+
  • Maven 2.2.1+

Moodle plugin

This download contains a Moodle plugin to provide learners with direct, single-sign-on access to a MyShowcase application.

  • Moodle 1.9.3+
  • Access to a live MyShowcase application
  • Please note, this plugin is not compatible with Moodle v2

Sakai tool

An alpha release Sakai implementation of the MyShowcase application built as a Sakai tool. This provides the core MyShowcase platform as an integrated component within Sakai, including single-sign-on access.

  • Tomcat 5.5.31+
  • Java 1.5 or 1.6
  • MySQL 5.1
  • MySQL Connector / J5.0.8+
  • Maven 2.2.1+
  • Sakai 2.6.2
